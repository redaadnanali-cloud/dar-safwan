# دار سفوان القرآنية - نسخة مكتملة

هذه الحزمة تضيف الملفات التي كانت ناقصة حول `index.html`:

- `manifest.json` حتى يظهر التطبيق كتطبيق قابل للتثبيت.
- `service-worker.js` حتى يعمل التطبيق بعد أول زيارة، ويحتفظ بالصفحة والأيقونات والملفات الأساسية.
- `icon-192.png` و`icon-512.png` كأيقونات للتطبيق.
- `firestore.rules` كقواعد حماية مقترحة لمزامنة تقدم كل مستخدم مع حسابه فقط.

## التشغيل الصحيح

لا تفتح `index.html` مباشرة من سطح المكتب إذا أردت اختبار التثبيت والعمل دون إنترنت، لأن Service Worker يحتاج `localhost` أو HTTPS.

من داخل هذا المجلد شغّل:

```powershell
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080/
```

## قبل النشر

1. ارفع كل ملفات هذا المجلد معًا، وليس `index.html` وحده.
2. في Firebase Console فعّل Authentication بالبريد وكلمة المرور.
3. ضع محتوى `firestore.rules` في Firestore Rules ثم انشرها.
4. أضف دومين النشر في Authentication > Settings > Authorized domains.
5. اختبر التطبيق على هاتف حقيقي، ثم جرّب وضع الطيران بعد أول فتح للتأكد من التخزين المؤقت.
